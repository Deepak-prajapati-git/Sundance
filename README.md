# Sundance Retreats & Holidays

A premium property showcase and travel experiences platform for Sundance Retreats.

## Features
- **Curated Property Collection**: Explore luxury stays like VN Villa and Hangout with high-quality galleries.
- **Interactive Property Modal**: Deep dive into property features, amenities, and download brochures.
- **Experiences Showcase**: Discover spiritual, heritage, and wellness journeys.
- **Premium UI/UX**: Fluid animations, custom cursor, and responsive design.

## Project Structure
- `index.html`: Main landing page with experiences and contact form.
- `properties.html`: Dedicated property listing and gallery.
- `style.css`: Centralized vanilla CSS for site-wide branding.
- `script.js`: Shared site-wide interactivity.
- `properties-modal.js`: Logic for the property detail modal.
- `Assets/`: Organized directory for images, videos, and property brochures.

## Hosting on GitHub Pages
1. Push this repository to GitHub.
2. Go to **Settings > Pages**.
3. Select the `main` branch and `/ (root)` folder.
4. Hit **Save**.

The `.nojekyll` file ensures all subdirectories (like `Assets/Properties/`) are served correctly.
